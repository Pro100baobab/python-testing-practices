# Техническое задание: Модели продуктов

## 1. Продукт «Рыба»

### Поля и ограничения

| Поле | Тип | Обязательное | Ограничения | Описание |
|------|-----|--------------|-------------|----------|
| `name` | строка | да | 1–100 символов, только буквы, цифры, пробелы, дефисы | Название рыбы |
| `kind` | строка (enum) | да | одно из: `salmon`, `cod`, `carp`, `perch`, `other` | Вид рыбы |
| `is_fresh` | булево | да | `true` или `false` | Свежая или замороженная |
| `weight_grams` | число (float) | да | > 0, ≤ 500000 | Вес в граммах |
| `price_per_kg` | число (float) | да | > 0, ≤ 10000 | Цена за килограмм |
| `country` | строка | нет | ≤ 100 символов, буквы (рус/лат), пробелы, дефисы | Страна происхождения |
| `quality_rating` | целое | нет | от 1 до 5 | Оценка качества |

### Вычисляемое свойство
- `total_price` — общая стоимость = (weight_grams / 1000) * price_per_kg

### Пример корректных данных
```json
{
  "name": "Семга атлантическая",
  "kind": "salmon",
  "is_fresh": true,
  "weight_grams": 1500.5,
  "price_per_kg": 1200.0,
  "country": "Норвегия",
  "quality_rating": 5
}
```

## 2. Продукт «Морс»

### Поля и ограничения

| Поле | Тип | Обязательное | Ограничения | Описание |
|------|-----|--------------|-------------|----------|
| name | строка | да | 1–100 символов, буквы, цифры, пробелы, дефисы | Название морса |
| berry_type | строка (enum) | да | cranberry, lingonberry, blueberry, raspberry, sea_buckthorn, other | Тип ягоды |
| volume_ml | число (float) | да | ≥ 0.1, ≤ 10000 | Объём в миллилитрах |
| has_sugar | булево | да | true или false | С сахаром или без |
| alcohol_percent | число (float) | нет (по умолч. 0.0) | 0.0–20.0 | Процент алкоголя |
| manufacturer | строка | нет | ≤ 100 символов, буквы, цифры, пробелы, дефисы, точки | Производитель |
| rating | целое | нет | от 1 до 5 | Оценка |

### Вычисляемое свойство

`energy_kcal` — энергетическая ценность:

- с сахаром: `round(volume_ml * 0.4, 1)`
- без сахара: `round(volume_ml * 0.1, 1)`

### Пример корректных данных

```json
{
  "name": "Клюквенный морс",
  "berry_type": "cranberry",
  "volume_ml": 500.0,
  "has_sugar": true,
  "alcohol_percent": 0.0,
  "manufacturer": "Ягодка",
  "rating": 5
}
```

### Общие требования

- Запрещены лишние поля (`additionalProperties: false`).
- Числовые поля не могут быть `NaN`, `Infinity` или отрицательным нулём.
- При попытке создать объект с некорректными данными выбрасывается исключение `ValueError`.
- Изменение данных после создания также проверяется на валидность.
